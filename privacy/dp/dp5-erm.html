01

Empirical Risk Minimization

A standard learning template.

ERM Setup

Data $X = ((x_1, y_1), \dots, (x_n, y_n))$, model $f_\theta : \mathcal{X} \to \hat{\mathcal{Y}}$, loss $\ell : \hat{\mathcal{Y}} \times \mathcal{Y} \to [0, \infty)$.

$$L(\theta, X) = \frac{1}{n}\sum_{i=1}^n \ell(f_\theta(x_i), y_i), \qquad R(\theta) = L(\theta, X) - \min_{\theta'} L(\theta', X).$$

Population versions assume $(X, Y) \sim P_X$: $L_p(\theta) = \mathbb{E}\bigl[\ell(f_\theta(X), Y)\bigr], \; R_p(\theta) = L_p(\theta) - \min_{\theta'} L_p(\theta')$.

Standard Assumptions

  • Constraint set. $\theta \in \mathcal{C}$ — e.g., $\|\theta\|_2^2 \le C$.
  • Bounded loss. $\ell(f_\theta(x), y) \in [0, \Delta]$ for all $\theta, x, y$.
  • Lipschitz gradient. $\|\nabla_\theta \ell(f_\theta(x), y)\| \le G$.
These bound the influence of any single record on the loss and gradients — exactly the quantities that drive DP sensitivity.
02

Exponential Mechanism for ERM

A private "soft-argmin" over the parameter space.

Exponential Mechanism for ERM

Sample $\Theta$ according to

$$\Pr[\Theta = \theta] \;\propto\; \exp\!\Bigl(-\frac{\varepsilon n}{2\Delta}\, L(\theta; X)\Bigr).$$
  • Sensitivity of $L(\theta; \cdot)$ is $\Delta/n$ — one record moves loss by at most $\Delta/n$.
  • The exponential mechanism with this score is $\varepsilon$-DP (Lecture 2).
  • Concentrates on low-loss $\theta$; sampling implementation can use MCMC.

Utility Theorem

Theorem.

Let $\mathcal{C}$ be the $\ell_2$-ball of radius $R$ in $\mathbb{R}^d$ and $\nabla\ell$ be $G$-Lipschitz. With $\Delta = GR$, the exponential mechanism returns $\hat\theta$ with

$$\mathbb{E}[L(\hat\theta, X)] \;\le\; \min_{\theta \in \mathcal{C}} L(\theta; X) \;+\; O\!\Bigl(\frac{d\, G\, R\, \log(\varepsilon n/d)}{\varepsilon n}\Bigr).$$

Excess risk scales as $\tilde O(d/(\varepsilon n))$ — the canonical DP-ERM bound.

Proof — Volume Ratio

Fix $r > 0$. Define

$$A_1 = \{\hat\theta \in \mathcal{C} : L(\hat\theta) \le L(\theta^*) + rG\}, \quad A_2 = \{\hat\theta \in \mathcal{C} : L(\hat\theta) > L(\theta^*) + rG + t\}.$$

By construction of the exponential distribution,

$$\frac{\Pr[\hat\theta \in A_2]}{\Pr[\hat\theta \in A_1]} \le \frac{\mathrm{Vol}(A_2)}{\mathrm{Vol}(A_1)} \exp\!\Bigl(-\frac{\varepsilon n t}{2 R G}\Bigr) \le \Bigl(\frac{R}{r/2}\Bigr)^d \exp\!\Bigl(-\frac{\varepsilon n t}{2 R G}\Bigr).$$

$r/2$ accounts for $\theta^*$ on the boundary of $\mathcal{C}$ — $A_1$ still contains a ball of radius $r/2$.

Proof — Tail-to-Expectation

Pick $t = \tfrac{2 R G}{\varepsilon n}\bigl(d \log(2R/r) + \log(1/\beta)\bigr)$ so that $\Pr[\hat\theta \in A_2] \le \beta$. Then

$$\mathbb{E}[L(\hat\theta) - L(\theta^*)] = \int_0^{rG} \Pr[L(\hat\theta) - L(\theta^*) \ge t]\, dt + \int_0^\infty \Pr[L(\hat\theta) - L(\theta^*) \ge rG + t]\, dt.$$

Bounding the second integral by the exponential tail and choosing $r = 2R d/(n\varepsilon)$:

$$\mathbb{E}[L(\hat\theta) - L(\theta^*)] = O\!\Bigl(\tfrac{d G R \log(\varepsilon n/d)}{\varepsilon n}\Bigr). \;\blacksquare$$
03

Advanced Composition

$k\varepsilon \to \sqrt{2k\ln(1/\delta')}\,\varepsilon$ when small $\delta$ is allowed.

Advanced Composition

Theorem (Dwork & Roth 2014).

If $\mathcal{M}_1, \dots, \mathcal{M}_k$ are $(\varepsilon, \delta)$-DP (possibly adaptive), then for any $\delta' > 0$ the $k$-fold composition is $(\varepsilon', k\delta + \delta')$-DP with

$$\varepsilon' \;=\; \sqrt{2 k \ln(1/\delta')}\, \varepsilon \;+\; k\, \varepsilon \cdot \tfrac{e^\varepsilon - 1}{e^\varepsilon + 1}.$$

For small $\varepsilon$: $\varepsilon' \approx \varepsilon \sqrt{2k\ln(1/\delta')}$ — privacy degrades like $\sqrt k$, not $k$.

Intuition

For adaptive $A = (A_1, \dots, A_k)$ with each $A_i$ being $(\varepsilon_i, \delta_i)$-DP:

$$\log \frac{\Pr[A(X) = (y_1, \dots, y_k)]}{\Pr[A(X') = (y_1, \dots, y_k)]} = \sum_{i=1}^k \log\frac{\Pr[A_i(X) = y_i \mid A^{i-1} = y^{i-1}]}{\Pr[A_i(X') = y_i \mid A^{i-1} = y^{i-1}]}.$$
  • Basic composition bounds each term separately by $\varepsilon$ ⇒ total $k\varepsilon$.
  • The sum of zero-mean bounded random variables concentrates at $\sqrt k$ ⇒ $\sqrt k\,\varepsilon$.
Tight bounds come from Rényi DP / Moments Accountant — see Lecture 6.
04

DP-SGD

Gradient descent with clipping and Gaussian noise.

Gradient Descent (GD)

Algorithm — GD  ($\Pi_\mathcal{C}$ = projection onto $\mathcal{C}$)
Initialize $\theta_0$
for $t = 1, \dots, T$:
  $g_t = \tfrac{1}{n}\sum_{i=1}^n \nabla_\theta \ell(\theta_{t-1}; x_i)$
  $\theta_t = \theta_{t-1} - \alpha\, g_t$
  $\theta_t \leftarrow \Pi_\mathcal{C}(\theta_t)$
return $\theta_T$

Convergence (convex, $G$-Lipschitz, diameter $R$).

With $\alpha = R / (G\sqrt T)$, $\mathcal{L}(\theta_T) \le \mathcal{L}(\theta^*) + RG/\sqrt T$.

SGD and Noisy GD

SGD
for $t = 1, \dots, T$:
  sample $x_i \in \mathcal{D}$
  $\tilde g_t = \nabla_\theta \ell(\theta_{t-1}; x_i)$
  $\theta_t = \theta_{t-1} - \alpha \tilde g_t$
  $\theta_t \leftarrow \Pi_\mathcal{C}(\theta_t)$
Noisy GD
for $t = 1, \dots, T$:
  $g_t = \tfrac{1}{n}\sum_i \nabla_\theta \ell(\theta_{t-1}; x_i)$
  $\tilde g_t = g_t + \mathcal{N}(0, \sigma^2 I)$
  $\theta_t = \theta_{t-1} - \alpha \tilde g_t$
  $\theta_t \leftarrow \Pi_\mathcal{C}(\theta_t)$

Both yield unbiased gradient estimates: $\mathbb{E}[\tilde g_t] = g_t$.

DP-SGD Algorithm

Algorithm — DP-SGD  (clip $\bar g_t$, add Gaussian)
Initialize $\theta_0$
for $t = 1, \dots, T$:
  sample $x_t \sim \mathcal{D}$ uniformly
  $g_t = \nabla_\theta \ell(\theta_{t-1}; x_t)$
  $\bar g_t = g_t \cdot \min\!\bigl(1, G/\|g_t\|_2\bigr)$  # clip
  $\tilde g_t = \bar g_t + \mathcal{N}(0, \sigma^2 I)$
  $\theta_t = \theta_{t-1} - \alpha\, \tilde g_t$
  $\theta_t \leftarrow \Pi_\mathcal{C}(\theta_t)$
return $\theta_T$
Abadi, Chu, Goodfellow, McMahan, Mironov, Talwar, Zhang, "Deep Learning with Differential Privacy", CCS 2016.

DP-SGD Privacy Guarantee

Theorem.

DP-SGD is $(\varepsilon, \delta)$-DP if the Gaussian noise satisfies

$$\sigma \;\ge\; \frac{2G\,\sqrt{2T\log(1/\delta)}}{n\,\varepsilon}.$$
  • $\sqrt T$ scaling comes from tight accounting (Moments Accountant / RDP).
  • Naïve advanced composition would add an extra $\sqrt{\log(T/\delta)}$ factor.

Why Subsampling Helps DP-SGD

  • Each step touches one record ⇒ sampling rate $q = 1/n$.
  • Per-step privacy cost is reduced from $(\varepsilon, \delta)$ to roughly $(\varepsilon/n,\, \delta/n)$ (amplification).
  • Over $T$ steps with tight accounting, total cost $\approx (\sqrt T\, \varepsilon/n,\; T\delta/n)$.
With $T = O(n^2)$ steps, total privacy remains $(O(\varepsilon),\, O(\delta))$ — many epochs are affordable.

DP-SGD Utility — Convex Case

Assume $\mathcal{L}$ convex, $G$-Lipschitz, $\mathcal{C}$ closed convex with diameter $R$. Since true gradient norms are $\le G$, setting clip threshold $= G$ avoids clipping. Then the noisy stochastic gradient satisfies:

  • Unbiased: $\mathbb{E}[\tilde g_t] = \nabla \mathcal{L}(\theta_{t-1})$.
  • Bounded variance: $\mathbb{E}[\|\tilde g_t\|^2] \le G^2 + d\sigma^2$ (Gaussian contributes $d\sigma^2$).

Theorem.

With $T = \varepsilon^2 n^2 / d$, $\alpha = \tfrac{1}{T}\tfrac{R\varepsilon n}{G\sqrt{d\log(1/\delta)}}$, average iterate $\bar\theta_T = \tfrac{1}{T}\sum_t \theta_t$ satisfies

$$\mathbb{E}[\mathcal{L}(\bar\theta_T)] - \mathcal{L}(\theta^*) = O\!\Bigl(\tfrac{R G \sqrt{d \log(1/\delta)}}{\varepsilon n}\Bigr).$$

Nonconvex Loss

  • Real ML losses (deep nets) are nonconvex — no global-optimality theorem in general.
  • DP-SGD (and SGD) still converge to a stationary point: $\nabla \mathcal{L}(\theta) = 0$.
  • Practical recipe: public pretrain → private fine-tune with DP-SGD.
DP-SGD is the de facto standard for differentially private deep learning today.

Recap

  • ERM + bounded-loss / Lipschitz-gradient assumptions ⇒ controlled DP sensitivity.
  • Exponential mechanism for ERM: excess risk $\tilde O(d/(\varepsilon n))$.
  • Advanced composition: $\sqrt k$ in place of $k$ when $\delta' > 0$.
  • DP-SGD: clip + Gaussian + tight accounting; $\sigma \ge 2G\sqrt{2T\log(1/\delta)}/(n\varepsilon)$.
  • Utility: $\tilde O(\sqrt d/(\varepsilon n))$ excess risk for convex losses.
