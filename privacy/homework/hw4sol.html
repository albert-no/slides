Data Privacy: Homework #4 â€” Solutions

  1. Neymanâ€“Pearson Lemma (Detailed Proof). Let $T \sim p_b$ on $(\mathcal{T}, \mathcal{B})$ under $H_b$ for $b\in\{0,1\}$, with densities w.r.t.\ a common dominating measure $\mu$. A test is $\phi : \mathcal{T} \to [0,1]$ with size $\alpha(\phi) = \mathbb{E}_{H_0}[\phi]$ and power $\beta(\phi) = \mathbb{E}_{H_1}[\phi]$. Fix $\alpha_0 \in (0,1)$. Prove:
    1. There exist $\tau \ge 0$ and $\gamma \in [0,1]$ such that $\phi^*(t) = \mathbf{1}[p_1(t) > \tau p_0(t)] + \gamma \mathbf{1}[p_1(t) = \tau p_0(t)]$ satisfies $\alpha(\phi^*) = \alpha_0$.
    2. For every test $\phi$ with $\alpha(\phi) \le \alpha_0$, $\beta(\phi) \le \beta(\phi^*)$.
    3. Any most-powerful level-$\alpha_0$ test agrees with $\phi^*$ $\mu$-a.e.\ on $\{p_1 \ne \tau p_0\}$.
    Solution.

    (a) Existence of $(\tau, \gamma)$. Define the likelihood ratio $\Lambda(t) = p_1(t)/p_0(t)$, with the convention $\Lambda = +\infty$ on $\{p_0 = 0, p_1 > 0\}$ (the test outputs $1$ there, contributing $0$ to $\alpha$ since $\Pr_{H_0}[\{p_0=0\}] = 0$). Consider the survival function $$G(\tau) \;=\; \Pr_{H_0}[\Lambda(T) > \tau], \quad \tau \in [0, \infty).$$ The sets $\{\Lambda > \tau\}$ are nested-decreasing in $\tau$, so $G$ is non-increasing. As $\tau_n \downarrow \tau$, $\{\Lambda > \tau_n\} \uparrow \{\Lambda > \tau\}$, so monotone convergence gives $G(\tau_n) \to G(\tau)$: $G$ is right-continuous. The left limit is $G(\tau-) = \Pr_{H_0}[\Lambda \ge \tau]$, and $G(\tau-) - G(\tau) = \Pr_{H_0}[\Lambda = \tau]$. Finally, $G(0-) \le 1$ and $G(\tau) \to 0$ as $\tau \to \infty$. Set $\tau^* \;=\; \inf\{s \ge 0 : G(s) \le \alpha_0\}$. By right-continuity and monotonicity, $G(\tau^*) \le \alpha_0 \le G(\tau^*-)$. If $G(\tau^*) = G(\tau^*-)$ then $\Pr[\Lambda = \tau^*] = 0$, and $\alpha_0 = G(\tau^*)$; any $\gamma$ works (take $\gamma = 0$). Otherwise set $$\gamma \;=\; \frac{\alpha_0 - G(\tau^*)}{G(\tau^*-) - G(\tau^*)} \;\in\; [0,1].$$ Then $$\alpha(\phi^*) = \Pr_{H_0}[\Lambda > \tau^*] + \gamma\,\Pr_{H_0}[\Lambda = \tau^*] = G(\tau^*) + \gamma(G(\tau^*-) - G(\tau^*)) = \alpha_0. \qquad\square$$

    (b) Optimality. Let $\phi$ be any test with $\alpha(\phi) \le \alpha_0$. We claim the pointwise inequality $$\big(\phi^*(t) - \phi(t)\big)\,\big(p_1(t) - \tau^* p_0(t)\big) \;\ge\; 0 \quad \text{for every } t.$$ Indeed, three cases:

    • If $p_1(t) > \tau^* p_0(t)$: $\phi^*(t) = 1 \ge \phi(t)$, and the second factor is positive â€” product $\ge 0$.
    • If $p_1(t) < \tau^* p_0(t)$: $\phi^*(t) = 0 \le \phi(t)$, and the second factor is negative â€” product $\ge 0$.
    • If $p_1(t) = \tau^* p_0(t)$: the second factor is $0$ â€” product is $0$.
    Integrate against $\mu$: $$0 \;\le\; \int (\phi^* - \phi)(p_1 - \tau^* p_0)\, \mathrm{d}\mu \;=\; \big(\beta(\phi^*) - \beta(\phi)\big) \;-\; \tau^*\big(\alpha(\phi^*) - \alpha(\phi)\big).$$ Since $\alpha(\phi^*) = \alpha_0 \ge \alpha(\phi)$ and $\tau^* \ge 0$, the second term is non-negative, hence $\beta(\phi^*) - \beta(\phi) \ge \tau^*(\alpha_0 - \alpha(\phi)) \ge 0$. $\quad\square$

    (c) Uniqueness. Suppose $\phi$ is also most powerful, so $\beta(\phi) = \beta(\phi^*)$ and $\alpha(\phi) \le \alpha_0 = \alpha(\phi^*)$. The chain of inequalities in (b) collapses: $$0 \;\le\; \int(\phi^* - \phi)(p_1 - \tau^* p_0)\,\mathrm{d}\mu \;=\; -\tau^*(\alpha_0 - \alpha(\phi)) \;\le\; 0.$$ Hence the integral equals $0$. The integrand is $\ge 0$ pointwise, so it must vanish $\mu$-a.e. On $\{p_1 > \tau^* p_0\}$, $p_1 - \tau^* p_0 > 0$, forcing $\phi^* - \phi = 0$ a.e., i.e., $\phi(t) = 1$. Likewise $\phi(t) = 0$ a.e.\ on $\{p_1 < \tau^* p_0\}$. So $\phi = \phi^*$ a.e.\ on $\{p_1 \ne \tau^* p_0\}$. $\quad\square$

  2. DP $\Rightarrow$ MIA Bounds. $M$ is $(\varepsilon,\delta)$-DP, $D_0 \sim D_1$ neighboring. For any test $\mathcal{A}$: $\mathrm{FPR} = \Pr[\mathcal{A}(M(D_0))=1]$, $\mathrm{TPR} = \Pr[\mathcal{A}(M(D_1))=1]$.
    1. $\mathrm{TPR} \le e^\varepsilon \mathrm{FPR} + \delta$ and $1 - \mathrm{FPR} \le e^\varepsilon(1 - \mathrm{TPR}) + \delta$.
    2. $\mathrm{Adv}(\mathcal{A}) \le e^\varepsilon - 1 + \delta$, and tighter $\mathrm{Adv} \le \frac{e^\varepsilon - 1 + 2\delta}{e^\varepsilon + 1}$.
    3. Numerical: $\varepsilon = 0.1, \delta = 10^{-5} \Rightarrow$ Adv $\lesssim 0.105$; $\varepsilon = 8 \Rightarrow$ vacuous.
    Solution.

    (a) Let $S = \{y : \mathcal{A}(y) = 1\}$ (if $\mathcal{A}$ is randomized, average over its internal randomness). Apply $(\varepsilon,\delta)$-DP to $S$ with $(D_1, D_0)$: $$\Pr[M(D_1) \in S] \;\le\; e^\varepsilon \Pr[M(D_0) \in S] + \delta \;\;\Longleftrightarrow\;\; \mathrm{TPR} \;\le\; e^\varepsilon \mathrm{FPR} + \delta.$$ Apply it to $S^c$ with $(D_0, D_1)$ (DP is symmetric in the pair): $$\Pr[M(D_0) \in S^c] \;\le\; e^\varepsilon \Pr[M(D_1) \in S^c] + \delta \;\;\Longleftrightarrow\;\; 1 - \mathrm{FPR} \;\le\; e^\varepsilon(1 - \mathrm{TPR}) + \delta.$$

    (b) Loose bound. From the first inequality of (a) and $\mathrm{FPR} \le 1$: $$\mathrm{Adv} = \mathrm{TPR} - \mathrm{FPR} \le (e^\varepsilon - 1)\mathrm{FPR} + \delta \le e^\varepsilon - 1 + \delta.$$

    Tighter (Kairouzâ€“Ohâ€“Viswanath). The feasible region is the polygon $$\mathrm{TPR} \le e^\varepsilon \mathrm{FPR} + \delta, \quad \mathrm{TPR} \ge 1 - e^\varepsilon(1 - \mathrm{FPR}) - \delta \cdot (-1) \;=\; e^{-\varepsilon}\mathrm{FPR} + 1 - e^{-\varepsilon} + \delta e^{-\varepsilon}.$$ (Rewrite the second inequality as $\mathrm{TPR} \ge 1 - e^{-\varepsilon}(1 - \mathrm{FPR}) + \delta e^{-\varepsilon}$ â€” but for the upper bound on Adv we use the symmetric form.) The maximum of $\mathrm{TPR} - \mathrm{FPR}$ occurs at the vertex where both inequalities are tight. Solving simultaneously, $$e^\varepsilon \mathrm{FPR} + \delta \;=\; 1 - e^{-\varepsilon}(1 - \mathrm{FPR}) + \delta e^{-\varepsilon},$$ $$(e^\varepsilon - e^{-\varepsilon})\mathrm{FPR} \;=\; (1 - e^{-\varepsilon})(1 - \delta),$$ $$\mathrm{FPR}^\star \;=\; \frac{(e^\varepsilon - 1)(1-\delta)}{e^{2\varepsilon} - 1} \;=\; \frac{1-\delta}{e^\varepsilon + 1}.$$ Then $\mathrm{TPR}^\star = e^\varepsilon \mathrm{FPR}^\star + \delta = \frac{e^\varepsilon + \delta}{e^\varepsilon + 1}$, so $$\mathrm{Adv}^\star \;=\; \mathrm{TPR}^\star - \mathrm{FPR}^\star \;=\; \frac{e^\varepsilon + \delta - (1 - \delta)}{e^\varepsilon + 1} \;=\; \frac{e^\varepsilon - 1 + 2\delta}{e^\varepsilon + 1}. \quad\square$$

    (c) Numerical. For $\varepsilon = 0.1, \delta = 10^{-5}$: loose bound $= e^{0.1} - 1 + 10^{-5} \approx 0.1052$. Tighter bound $= (0.1052 + 2 \cdot 10^{-5})/2.1052 \approx 0.0500$. For $\varepsilon = 8$: $e^8 \approx 2981$, so loose bound is $\approx 2980$ and tighter bound is $\approx 1$ â€” both useless, since advantage is automatically $\le 1$.

  3. Bayes-Optimal Membership Test. $B \sim \mathrm{Bern}(\pi)$, $T \mid B = b \sim p_b$. Want to compute MAP rule and connect to NP.
    1. Posterior of $B$ given $T = t$.
    2. MAP rule = LR test with threshold $(1-\pi)/\pi$; connect to Problem 1.
    3. Gaussian losses $\Rightarrow$ single-threshold test $\hat B = \mathbf{1}[\ell \le c]$ with $c = (\mu_{\mathrm{in}} + \mu_{\mathrm{out}})/2$ for $\pi = 1/2$.
    Solution.

    (a) By Bayes' rule, $$\Pr[B = 1 \mid T = t] \;=\; \frac{\pi\, p_1(t)}{\pi\, p_1(t) + (1-\pi)\, p_0(t)}.$$

    (b) The Bayes (0-1 loss) rule outputs $\hat B = 1$ iff $\Pr[B=1|T=t] \ge \Pr[B=0|T=t]$, i.e., $$\pi\, p_1(t) \;\ge\; (1-\pi)\, p_0(t) \;\;\Longleftrightarrow\;\; \Lambda(t) := \frac{p_1(t)}{p_0(t)} \;\ge\; \frac{1-\pi}{\pi}.$$ By Problem 1, this is a (non-randomized) Neymanâ€“Pearson test at the unique level $\alpha_0$ such that the threshold $\tau = (1-\pi)/\pi$ is the NP threshold. Changing the prior $\pi$ only changes the threshold; the form of the optimal MIA is always an LR test.

    (c) Gaussian case. With shared variance $\sigma^2$, the log-LR is $$\log\Lambda(\ell) \;=\; -\frac{(\ell - \mu_{\mathrm{in}})^2 - (\ell - \mu_{\mathrm{out}})^2}{2\sigma^2} \;=\; \frac{(\mu_{\mathrm{out}} - \mu_{\mathrm{in}})(\mu_{\mathrm{in}} + \mu_{\mathrm{out}} - 2\ell)}{2\sigma^2}.$$ Since $\mu_{\mathrm{out}} > \mu_{\mathrm{in}}$, this is a strictly decreasing affine function of $\ell$. So $\log\Lambda \ge \log\big((1-\pi)/\pi\big)$ iff $$\ell \;\le\; c \;=\; \frac{\mu_{\mathrm{in}} + \mu_{\mathrm{out}}}{2} \;-\; \frac{\sigma^2}{\mu_{\mathrm{out}} - \mu_{\mathrm{in}}}\,\log\!\frac{1-\pi}{\pi}.$$ For $\pi = 1/2$ the log term vanishes, giving $c = (\mu_{\mathrm{in}} + \mu_{\mathrm{out}})/2$. $\quad\square$

  4. Generalization Gap as a Leakage Lower Bound (Yeom). Bounded loss $\ell \in [0, B]$. Threshold attack $\mathcal{A}_\tau(z) = \mathbf{1}[\ell \le \tau]$.
    1. $\mathrm{Adv}(\mathcal{A}_\tau) = F_1(\tau) - F_0(\tau)$.
    2. $\int_0^B (F_1 - F_0)\, \mathrm{d}t = \Delta$.
    3. $\sup_\tau \mathrm{Adv}(\mathcal{A}_\tau) \ge \Delta/B$.
    4. Gaussian: optimal $\tau^\star = (R_{\mathrm{train}} + R_{\mathrm{pop}})/2$, $\mathrm{Adv}^\star = 2\Phi(\Delta/(2\sigma)) - 1$, with linear regime $\Delta/(\sigma\sqrt{2\pi})$.
    Solution.

    (a) By definitions, $\mathrm{TPR} = \Pr[\mathcal{A}_\tau = 1 \mid b = 1] = \Pr[\ell \le \tau \mid b = 1] = F_1(\tau)$, $\mathrm{FPR} = F_0(\tau)$. Subtract.

    (b) Layer cake. For a non-negative r.v. $X \in [0, B]$, integration by parts (or Fubini on $\{(t, X) : 0 \le t < X\}$) gives $$\mathbb{E}[X] \;=\; \int_0^B \Pr[X > t]\, \mathrm{d}t \;=\; \int_0^B (1 - F_X(t))\, \mathrm{d}t.$$ Apply to $\ell$ under each hypothesis: $$R_{\mathrm{pop}} - R_{\mathrm{train}} \;=\; \int_0^B \big[(1 - F_0(t)) - (1 - F_1(t))\big]\, \mathrm{d}t \;=\; \int_0^B (F_1(t) - F_0(t))\, \mathrm{d}t \;=\; \Delta.$$

    (c) Mean-value extraction. Let $h(t) = F_1(t) - F_0(t)$. If $h(t) < \Delta/B$ for every $t \in [0, B]$, then $\int h\,\mathrm{d}t < B \cdot (\Delta/B) = \Delta$, contradicting (b). So there exists $\tau^\star$ with $h(\tau^\star) \ge \Delta/B$, hence $\mathrm{Adv}(\mathcal{A}_{\tau^\star}) \ge \Delta/B$. The generalization gap is a lower bound on the leakage of the best threshold attack, not an upper bound.

    (d) Gaussian. With $\mu_1 = R_{\mathrm{train}}, \mu_0 = R_{\mathrm{pop}}$, $F_b(t) = \Phi((t - \mu_b)/\sigma)$, so $$h(\tau) \;=\; \Phi\!\left(\tfrac{\tau - R_{\mathrm{train}}}{\sigma}\right) - \Phi\!\left(\tfrac{\tau - R_{\mathrm{pop}}}{\sigma}\right).$$ Setting $h'(\tau) = 0$ gives $\phi((\tau - R_{\mathrm{train}})/\sigma) = \phi((\tau - R_{\mathrm{pop}})/\sigma)$, so $|\tau - R_{\mathrm{train}}| = |\tau - R_{\mathrm{pop}}|$, i.e., $\tau^\star = (R_{\mathrm{train}} + R_{\mathrm{pop}})/2$. Then $\tau^\star - R_{\mathrm{train}} = \Delta/2$ and $\tau^\star - R_{\mathrm{pop}} = -\Delta/2$, so $$\mathrm{Adv}^\star \;=\; \Phi(\Delta/(2\sigma)) - \Phi(-\Delta/(2\sigma)) \;=\; 2\Phi(\Delta/(2\sigma)) - 1.$$ For small $x = \Delta/(2\sigma)$, $\Phi(x) \approx 1/2 + x\phi(0) = 1/2 + x/\sqrt{2\pi}$, so $\mathrm{Adv}^\star \approx 2x/\sqrt{2\pi} = \Delta/(\sigma\sqrt{2\pi})$. Larger $\sigma$ (noisier loss distributions) flattens the gap and reduces leakage â€” the same idea behind DP noise injection. $\quad\square$

  5. LiRA Closed Form and Decision Boundary. $\ell \mid (z \in D) \sim \mathcal{N}(\mu_{\mathrm{in}}, \sigma_{\mathrm{in}}^2)$, $\ell \mid (z \notin D) \sim \mathcal{N}(\mu_{\mathrm{out}}, \sigma_{\mathrm{out}}^2)$.
    1. Log-LR closed form.
    2. Structure of decision region for $\sigma_{\mathrm{in}} \ne \sigma_{\mathrm{out}}$.
    3. Homoscedastic reduction to single threshold.
    4. Homoscedastic ROC: $\mathrm{TPR}(\alpha) = \Phi(\Phi^{-1}(\alpha) + (\mu_{\mathrm{out}} - \mu_{\mathrm{in}})/\sigma)$.
    Solution.

    (a) The Gaussian log-density is $\log\mathcal{N}(\ell;\mu,\sigma^2) = -\frac{(\ell-\mu)^2}{2\sigma^2} - \frac{1}{2}\log(2\pi\sigma^2)$. Subtracting, $$\log\Lambda(\ell) \;=\; \frac{(\ell - \mu_{\mathrm{out}})^2}{2\sigma_{\mathrm{out}}^2} - \frac{(\ell - \mu_{\mathrm{in}})^2}{2\sigma_{\mathrm{in}}^2} + \log\frac{\sigma_{\mathrm{out}}}{\sigma_{\mathrm{in}}}.$$

    (b) Rearranging, $\log\Lambda(\ell)$ is a quadratic in $\ell$ with leading coefficient $$A \;=\; \frac{1}{2}\left(\frac{1}{\sigma_{\mathrm{out}}^2} - \frac{1}{\sigma_{\mathrm{in}}^2}\right).$$

    • If $\sigma_{\mathrm{in}} < \sigma_{\mathrm{out}}$ (members concentrated): $A < 0$, $\log\Lambda$ is concave in $\ell$, so $\{\log\Lambda > \log\tau\}$ is an interval $(c_-, c_+)$ (possibly empty, possibly a half-line).
    • If $\sigma_{\mathrm{in}} > \sigma_{\mathrm{out}}$: $A > 0$, $\log\Lambda$ is convex, $\{\log\Lambda > \log\tau\}$ is the complement of an interval: $\{\ell < c_-\} \cup \{\ell > c_+\}$.
    • If $\sigma_{\mathrm{in}} = \sigma_{\mathrm{out}}$: $A = 0$, $\log\Lambda$ is linear, region is a half-line.

    (c) Homoscedastic. With $\sigma_{\mathrm{in}} = \sigma_{\mathrm{out}} = \sigma$ the quadratic terms cancel and $$\log\Lambda(\ell) \;=\; \frac{(\ell - \mu_{\mathrm{out}})^2 - (\ell - \mu_{\mathrm{in}})^2}{2\sigma^2} \;=\; \frac{(\mu_{\mathrm{out}} - \mu_{\mathrm{in}})(\mu_{\mathrm{in}} + \mu_{\mathrm{out}} - 2\ell)}{2\sigma^2},$$ a strictly decreasing affine function of $\ell$ (since $\mu_{\mathrm{out}} > \mu_{\mathrm{in}}$). Hence $\log\Lambda \ge \log\tau$ iff $\ell \le c$, where $$c \;=\; \frac{\mu_{\mathrm{in}} + \mu_{\mathrm{out}}}{2} \;-\; \frac{\sigma^2 \log\tau}{\mu_{\mathrm{out}} - \mu_{\mathrm{in}}}.$$

    (d) Homoscedastic ROC. $\mathrm{FPR}(c) = \Pr_{\mathrm{out}}[\ell \le c] = \Phi((c - \mu_{\mathrm{out}})/\sigma)$. Solve for $c$ at $\mathrm{FPR} = \alpha$: $c = \mu_{\mathrm{out}} + \sigma\Phi^{-1}(\alpha)$. Substitute: $$\mathrm{TPR}(\alpha) \;=\; \Pr_{\mathrm{in}}[\ell \le c] \;=\; \Phi\!\left(\frac{c - \mu_{\mathrm{in}}}{\sigma}\right) \;=\; \Phi\!\left(\Phi^{-1}(\alpha) + \frac{\mu_{\mathrm{out}} - \mu_{\mathrm{in}}}{\sigma}\right).$$ The MIA's discrimination is controlled entirely by the standardized "gap" $(\mu_{\mathrm{out}} - \mu_{\mathrm{in}})/\sigma$. $\quad\square$

  6. Concrete MIA on a Gaussian Mean Model. $\theta = \bar z + W$, $W \sim \mathcal{N}(0, \rho^2)$, $z_i \sim \mathcal{N}(0,1)$.
    1. Conditional densities of $\theta$ given $z^*$ under each $b$.
    2. Log-LR is quadratic in $\theta$.
    3. DP bound on the optimal MIA advantage.
    Solution.

    (a) Under $b=0$: $z^*$ is drawn fresh from $\mathcal{P}$, independent of $D$. $\bar z = (1/n)\sum_{i=1}^n z_i \sim \mathcal{N}(0, 1/n)$. So $\theta \sim \mathcal{N}(0, 1/n + \rho^2)$, independent of $z^*$.

    Under $b=1$: $z^* = z_1$. Write $\bar z = z^*/n + (1/n)\sum_{i=2}^n z_i$. The sum on the right is independent of $z^*$ with distribution $\mathcal{N}(0, (n-1))$, so $(1/n)\sum_{i \ge 2} z_i \sim \mathcal{N}(0, (n-1)/n^2)$. Hence $$\theta \mid z^* \;\sim\; \mathcal{N}\!\left(\tfrac{z^*}{n},\; \tfrac{n-1}{n^2} + \rho^2\right).$$

    (b) Log-LR. Let $V_1 = (n-1)/n^2 + \rho^2$ and $V_0 = 1/n + \rho^2$. Note $V_1 < V_0$ since $(n-1)/n^2 = 1/n - 1/n^2$. Then $$\log\Lambda(\theta, z^*) \;=\; -\frac{(\theta - z^*/n)^2}{2 V_1} + \frac{\theta^2}{2 V_0} + \tfrac{1}{2}\log\frac{V_0}{V_1}.$$ Expanding, $$\log\Lambda \;=\; \left(\frac{1}{2 V_0} - \frac{1}{2 V_1}\right)\theta^2 \;+\; \frac{z^*}{n V_1}\,\theta \;+\; \text{const}(z^*).$$ The coefficient of $\theta^2$ is negative ($V_1 < V_0$ implies $1/V_0 < 1/V_1$), so $\log\Lambda$ is a concave (downward) parabola in $\theta$, and the "declare member" region is an interval around the maximum.

    (c) DP bound. Under one-element neighbors with $|z_i|, |z_i'| \le 1$, the $L_2$-sensitivity of $\bar z$ is $|z_i - z_i'|/n \le 2/n$. The Gaussian-mechanism guarantee gives $(\varepsilon,\delta)$-DP for $$\rho \;\ge\; \frac{2}{n\varepsilon}\sqrt{2\log(1.25/\delta)}.$$ By Problem 2(b) the optimal MIA advantage is bounded by $$\mathrm{Adv}^\star \;\le\; e^\varepsilon - 1 + \delta.$$ For $\varepsilon = 0.1, \delta = 10^{-5}$: $\mathrm{Adv}^\star \le e^{0.1} - 1 + 10^{-5} \approx 0.1052$. The optimal MIA can correctly distinguish member from non-member with advantage at most $\approx 10.5\%$ above chance. For $\varepsilon = 1$, the bound is $e - 1 \approx 1.718 > 1$, which is vacuous because advantage is automatically in $[0, 1]$. Past $\varepsilon \approx \log 2 \approx 0.693$, the loose bound says nothing. The tighter Kairouz form is more informative in that regime. $\quad\square$

  7. Influence-Function Unlearning (Closed Form). $L$ $\lambda$-strongly convex, $G$-Lipschitz. Show $\theta_{-z} - \theta_o \approx \frac{1}{n}H^{-1}\nabla\ell(\theta_o; z)$ and bound the deletion sensitivity.
    1. Implicit-function derivative.
    2. First-order Newton-step approximation.
    3. $L_2$-sensitivity bound $\le 2G/(\lambda n)$.
    Solution.

    (a) IFT derivation. Define the path $L_t(\theta) = L(\theta; D) - (t/n)\,\ell(\theta; z)$ for $t \in [0, 1]$. By strong convexity, the unique minimizer $\theta(t) = \arg\min L_t$ exists and satisfies the first-order condition $$F(\theta(t), t) \;:=\; \nabla_\theta L(\theta(t); D) - \frac{t}{n}\,\nabla_\theta \ell(\theta(t); z) \;=\; 0.$$ Differentiate w.r.t.\ $t$: $$\underbrace{\Big[\nabla^2 L(\theta(t); D) - \tfrac{t}{n}\nabla^2 \ell(\theta(t); z)\Big]}_{=: H_t}\,\theta'(t) \;-\; \frac{1}{n}\,\nabla_\theta \ell(\theta(t); z) \;=\; 0.$$ At $t = 0$, $H_0 = \nabla^2 L(\theta_o; D) = H_{\theta_o}$, and $\theta(0) = \theta_o$. Solving, $$\theta'(0) \;=\; \frac{1}{n}\, H_{\theta_o}^{-1}\,\nabla_\theta \ell(\theta_o; z).$$ ($H_{\theta_o} \succeq \lambda I$ is invertible.)

    (b) Taylor expansion. At $t = 1$, $L_1(\theta) = (1 - 1/n) L(\theta; D \setminus \{z\}) \cdot$ (up to a positive scale), so $\theta(1) = \theta_{-z}$. First-order Taylor: $$\theta_{-z} - \theta_o \;=\; \theta(1) - \theta(0) \;\approx\; \theta'(0)\cdot 1 \;=\; \frac{1}{n}\, H_{\theta_o}^{-1}\,\nabla_\theta \ell(\theta_o; z).$$ Thus the unlearned parameter is $\theta_u = \theta_o + \frac{1}{n} H_{\theta_o}^{-1} \nabla_\theta \ell(\theta_o; z)$, exactly the Newton step in the slides.

    (c) Sensitivity. For two candidate deleted points $z, z'$: $$\theta_u(D, z) - \theta_u(D, z') \;=\; \frac{1}{n}\, H_{\theta_o}^{-1}\,\big(\nabla\ell(\theta_o; z) - \nabla\ell(\theta_o; z')\big).$$ Take norms: $\|H_{\theta_o}^{-1}\|_{\mathrm{op}} \le 1/\lambda$ by $\lambda$-strong convexity, and each gradient has norm $\le G$ by Lipschitz, so $$\|\theta_u(D, z) - \theta_u(D, z')\| \;\le\; \frac{1}{\lambda n}\,(G + G) \;=\; \frac{2G}{\lambda n}. \quad\square$$ This is the $L_2$-sensitivity needed to calibrate Gaussian noise in the certified-unlearning recipe.

  8. Certified $(\varepsilon,\delta)$-Unlearning by Gaussian Noise. $\mathcal{U}(D, z) = \theta_o + \frac{1}{n}H^{-1}\nabla\ell + N$ with $N \sim \mathcal{N}(0, \sigma^2 I)$. Second-order error $\eta \le C_2 G^2/(\lambda^2 n^2)$.
    1. Total $L_2$-sensitivity from retrain is $\le \eta$.
    2. Apply Gaussian mechanism; required $\sigma \ge \frac{C_2 G^2}{\lambda^2 n^2 \varepsilon}\sqrt{2\log(1.25/\delta)}$.
    3. Compare scaling with DP-of-training.
    Solution.

    (a) Sensitivity. Define the first-order unlearned parameter $\theta_u^{\mathrm{1st}} = \theta_o + \frac{1}{n} H^{-1} \nabla\ell(\theta_o; z)$, so $\mathcal{U}(D, z) = \theta_u^{\mathrm{1st}} + N$. The exact retrain is $\theta_{-z} = A(D \setminus \{z\})$. The hypothesis bounds $$\|\theta_u^{\mathrm{1st}} - \theta_{-z}\| \;\le\; \eta \;\le\; \frac{C_2 G^2}{\lambda^2 n^2}.$$ Therefore the deterministic part of $\mathcal{U}$ differs from $A(D \setminus\{z\})$ by at most $\eta$ in $L_2$.

    (b) Gaussian-mechanism bound. The standard Gaussian mechanism: if $f, g : \mathcal{X}^n \to \mathbb{R}^d$ satisfy $\|f(X) - g(X)\| \le \eta$ for all $X$, then the distributions of $f(X) + N$ and $g(X) + N$ are $(\varepsilon, \delta)$-indistinguishable when $\sigma \ge \eta \sqrt{2\log(1.25/\delta)}/\varepsilon$ (this is the "analytic" Gaussian guarantee from the slides; for the classical version one needs $\varepsilon \le 1$, but the conclusion is unchanged in scaling). Apply with $f = \theta_u^{\mathrm{1st}}$ and $g = \theta_{-z}$. The required noise is $$\sigma \;\ge\; \frac{\eta}{\varepsilon}\sqrt{2\log(1.25/\delta)} \;\le\; \frac{C_2\, G^2}{\lambda^2 n^2\, \varepsilon}\,\sqrt{2\log(1.25/\delta)},$$ matching Guo et al.\ (2020).

    (c) Scaling. The unlearning noise scales as $1/n^2$, whereas DP of the trained model scales as $1/n$ (sensitivity of a single training point in the loss/parameter is $O(1/n)$, requiring $\sigma_{\mathrm{DP}} \propto 1/(n\varepsilon)$). The unlearning task is easier for large $n$ because the influence of a single point on the optimum is itself $O(1/n)$ â€” so removing it perturbs the parameter by $O(1/n)$ in expectation and the residual after the Newton correction is $O(1/n^2)$, much smaller than what full DP would need to mask. $\quad\square$

  9. NPO Has a Bounded Gradient; GA Does Not.
    1. $\|\nabla_\theta \mathcal{L}_{\mathrm{GA}}\| \to \infty$ as $\pi_\theta \to 0$.
    2. $\partial\mathcal{L}_{\mathrm{NPO}}/\partial u = (2/\beta)\sigma(u) \in (0, 2/\beta)$.
    3. NPO gradient is uniformly bounded in $\theta$; sigmoid saturation makes the "safe" step.
    Solution.

    (a) GA gradient blowup. By definition $$\nabla_\theta \mathcal{L}_{\mathrm{GA}}(\theta) \;=\; \mathbb{E}_{(x,y)\sim \mathcal{D}_F}\!\big[\nabla_\theta \log\pi_\theta(y\mid x)\big] \;=\; \mathbb{E}\!\left[\frac{\nabla_\theta \pi_\theta(y \mid x)}{\pi_\theta(y \mid x)}\right].$$ If a forget sample has $\pi_\theta(y \mid x) \to 0$ with $\|\nabla_\theta \pi_\theta(y\mid x)\|$ bounded away from $0$ in some coordinate, the integrand blows up. Therefore $\|\nabla_\theta \mathcal{L}_{\mathrm{GA}}\| \to \infty$, and a single gradient-ascent step takes an arbitrarily large jump in $\theta$, pushing the model off the data manifold and destroying fluency on unrelated inputs. This is the "GA collapse" observed by Jang et al.

    (b) NPO bounded gradient. Let $u(\theta) = \beta\log(\pi_\theta(y\mid x)/\pi_{\mathrm{ref}}(y\mid x))$ (per-sample log-ratio scaled by $\beta$). Then $$\mathcal{L}_{\mathrm{NPO}}(\theta) \;=\; \frac{2}{\beta}\,\mathbb{E}[\log(1 + e^{u(\theta)})], \qquad \frac{\partial \mathcal{L}_{\mathrm{NPO}}}{\partial u} \;=\; \frac{2}{\beta}\cdot \frac{e^u}{1 + e^u} \;=\; \frac{2}{\beta}\,\sigma(u),$$ with $\sigma(u) \in (0, 1)$, so $\partial \mathcal{L}_{\mathrm{NPO}}/\partial u \in (0, 2/\beta)$. By the chain rule, $$\nabla_\theta \mathcal{L}_{\mathrm{NPO}} \;=\; \mathbb{E}\!\left[\frac{\partial \mathcal{L}}{\partial u}\,\nabla_\theta u\right] \;=\; \mathbb{E}\!\left[\frac{2}{\beta}\sigma(u)\cdot \beta\,\nabla_\theta \log\pi_\theta\right] \;=\; 2\,\mathbb{E}\!\left[\sigma(u)\,\nabla_\theta \log\pi_\theta\right],$$ hence the bound $\|\nabla_\theta \mathcal{L}_{\mathrm{NPO}}\| \le 2\,\mathbb{E}\!\left[\|\nabla_\theta \log\pi_\theta\|\right]$.

    (c) Where saturation comes from. As unlearning progresses on a forget sample, $\pi_\theta(y\mid x) \to 0$ â€” but then $u \to -\infty$ and $\sigma(u) \to 0$, so the per-sample contribution $\sigma(u)\nabla_\theta\log\pi_\theta$ shrinks even though $\nabla_\theta \log\pi_\theta$ itself blows up like $1/\pi_\theta$. More precisely, $\sigma(u) \sim e^u = (\pi_\theta/\pi_{\mathrm{ref}})^\beta$ for $u \to -\infty$, so $$\sigma(u)\,\nabla_\theta \log\pi_\theta \;\sim\; \frac{\pi_\theta^{\beta - 1}}{\pi_{\mathrm{ref}}^\beta}\,\nabla_\theta \pi_\theta,$$ which stays bounded for $\beta \ge 1$ (and remains tame in practice for smaller $\beta$ when $\pi_\theta$ shrinks faster than the gradient grows). Operationally: once a forget sample's likelihood is small enough, NPO essentially stops updating $\theta$ on it. GA never stops, which is why it collapses; NPO's sigmoid saturation is the entire fix. $\quad\square$

  10. Black-Box Unlearning Audits Require Many Queries. $\mathrm{TV}(P_0, P_1) \le \eta$. Auditor sees $q$ i.i.d.\ samples and outputs $\hat b$.
    1. Le Cam: $\Pr_0[\hat b\ne 0] + \Pr_1[\hat b\ne 1] \ge 1 - \mathrm{TV}(P_0^{\otimes q}, P_1^{\otimes q})$.
    2. Hybrid: $\mathrm{TV}(P_0^{\otimes q}, P_1^{\otimes q}) \le q\eta$.
    3. Sample complexity: $q \ge (1 - 2\alpha)/\eta$ to guarantee error sum $\le 2\alpha$.
    Solution.

    (a) Le Cam lower bound. The MIA auditor's decision $\hat b$ is a deterministic function (or randomized rule) of the $q$ observations; identify $\hat b$ with a test $\psi \in [0,1]$ representing the conditional probability of declaring $b = 1$. Then for $P = P_1^{\otimes q}$ and $Q = P_0^{\otimes q}$, $$\Pr_0[\hat b \ne 0] + \Pr_1[\hat b \ne 1] \;=\; \mathbb{E}_Q[\psi] + \mathbb{E}_P[1 - \psi] \;=\; 1 - \big(\mathbb{E}_P[\psi] - \mathbb{E}_Q[\psi]\big).$$ By the variational definition of total variation, $\mathbb{E}_P[\psi] - \mathbb{E}_Q[\psi] \le \mathrm{TV}(P, Q)$. Therefore $$\Pr_0[\hat b \ne 0] + \Pr_1[\hat b \ne 1] \;\ge\; 1 - \mathrm{TV}(P_1^{\otimes q}, P_0^{\otimes q}).$$

    (b) Hybrid (telescoping) bound. Construct intermediate distributions $H_k = P_0^{\otimes k} \otimes P_1^{\otimes (q-k)}$ for $k = 0, \ldots, q$, so $H_q = P_0^{\otimes q}$, $H_0 = P_1^{\otimes q}$. By the triangle inequality for TV, $$\mathrm{TV}(P_0^{\otimes q}, P_1^{\otimes q}) \;\le\; \sum_{k=1}^{q} \mathrm{TV}(H_k, H_{k-1}).$$ Each pair $H_k, H_{k-1}$ differs only in coordinate $k$: $H_k$ has $P_0$, $H_{k-1}$ has $P_1$. By the product structure (or directly by a coupling on that coordinate), $$\mathrm{TV}(H_k, H_{k-1}) \;=\; \mathrm{TV}(P_0, P_1) \;\le\; \eta.$$ Summing, $\mathrm{TV}(P_0^{\otimes q}, P_1^{\otimes q}) \le q\eta$.

    (c) Sample complexity. To guarantee the error sum $\le 2\alpha$, combining (a) and (b), $$2\alpha \;\ge\; \Pr_0[\hat b\ne 0] + \Pr_1[\hat b\ne 1] \;\ge\; 1 - q\eta \;\;\Longrightarrow\;\; q \;\ge\; \frac{1 - 2\alpha}{\eta}.$$ For $\alpha = 0.1$ this needs $q \ge 0.8/\eta$. Tighter bounds via Pinsker's inequality and the Hellinger affinity $\mathrm{TV}(P^{\otimes q}, Q^{\otimes q})^2 \le 2 q\, \mathrm{KL}(P\|Q)$ give the well-known $q = \Omega(1/\eta^2)$ scaling (and matching $\chi^2$ form behind Pawelczyk et al.\ 2024).

    Implication for unlearning: if the unlearned model $\theta_u$ differs from the original $\theta_o$ in output distribution by a small TV $\eta$, then a black-box auditor with sub-$1/\eta$ queries cannot reliably distinguish them. Single-prompt audits â€” the common regulatory-checkbox approach â€” fundamentally cannot certify that unlearning has happened; the auditor must use distributional probes. $\quad\square$
