IB 1 — Notes: The Information Bottleneck

Companion notes for ib1-foundations.html. Full derivations and context that don't fit on the slides.

§01 — Compression for relevance

Why IB, not just rate-distortion?

Classical rate-distortion theory (covered in lossy/lossy1-foundations.html) measures how well $T$ reconstructs $X$ under a distortion $d(x,t)$. The distortion is defined on the input space and doesn't reference any target $Y$.

The IB shifts the question: we don't care how well $T$ reconstructs $X$; we care how well $T$ predicts $Y$. Two inputs $x_1, x_2$ that induce the same $p(y|x)$ should map to the same cluster, even if they're far apart in input space.

Sufficiency and the IB

A sufficient statistic $T = S(X)$ for $Y$ satisfies $p(y|x) = p(y|S(x))$ for all $x$, i.e., $Y - S(X) - X$ is also Markov (in addition to $Y - X - S(X)$). For a sufficient statistic, $I(T;Y) = I(X;Y)$ — no information is lost.

The IB at $\beta \to \infty$ seeks the minimal sufficient statistic: the most compressed $T$ with $I(T;Y) = I(X;Y)$. For finite $\beta$, the IB relaxes sufficiency — it tolerates some loss in $I(T;Y)$ in exchange for compression.

Connection to Fano's inequality

Recall from entropy/entropy2-joint-mi-fano.html that Fano's inequality gives:

$$P_e \ge 1 - \frac{I(X;T) + 1}{\log|\mathcal{X}|}.$$

The IB can be seen as a refinement: instead of bounding error from rate, it directly trades rate against relevance (MI with the target $Y$).

§02 — Self-consistent equations

Full variational derivative

We expand the $I(X;T)$ derivative in detail. Write:

$$I(X;T) = \sum_{x,t} p(x)\, p(t|x)\, \log\frac{p(t|x)}{p(t)}.$$

Taking $\frac{\partial}{\partial p(t|x)}$ for a specific $(x,t)$ pair:

$$\frac{\partial}{\partial p(t|x)}\left[p(x)\, p(t|x)\,\log\frac{p(t|x)}{p(t)}\right] = p(x)\left(\log\frac{p(t|x)}{p(t)} + 1\right).$$

We also need the effect through $p(t) = \sum_{x'} p(x')\, p(t|x')$, so $\frac{\partial p(t)}{\partial p(t|x)} = p(x)$. This gives an additional contribution from all $(x', t)$ terms:

$$-\sum_{x'} p(x')\, p(t|x')\,\frac{p(x)}{p(t)} = -p(x).$$

Total: $p(x)\left(\log\frac{p(t|x)}{p(t)} + 1 - 1\right) = p(x)\log\frac{p(t|x)}{p(t)}$.

The $I(T;Y)$ derivative

Write $I(T;Y) = \sum_{t,y} p(t,y)\log\frac{p(y|t)}{p(y)}$, where $p(t,y) = \sum_{x'} p(x')\, p(t|x')\, p(y|x')$.

Direct contribution from $\frac{\partial p(t,y)}{\partial p(t|x)} = p(x)\, p(y|x)$:

$$\sum_y p(x)\, p(y|x)\,\log\frac{p(y|t)}{p(y)}.$$

The contribution from $p(y|t)$'s implicit dependence on $p(t|x)$ telescopes to zero (analogous to the $I(X;T)$ calculation). The cancellation follows from $\sum_y \frac{\partial}{\partial p(t|x)}\left[p(t,y)\log p(y|t)\right]$ simplifying via $p(y|t) = p(t,y)/p(t)$.

KL decomposition step

The key algebraic step in the proof:

$$\sum_y p(y|x)\log\frac{p(y|t)}{p(y)} = \sum_y p(y|x)\log\frac{p(y|x)}{p(y)} - \sum_y p(y|x)\log\frac{p(y|x)}{p(y|t)}$$
$$= D_{\mathrm{KL}}(p(y|x)\|p(y)) - D_{\mathrm{KL}}(p(y|x)\|p(y|t)).$$

Since $D_{\mathrm{KL}}(p(y|x)\|p(y))$ is independent of $t$, it is absorbed into the normalization constant $Z(x,\beta)$.

Blahut-Arimoto connection

The iterative algorithm to solve the self-consistent equations mirrors the Blahut-Arimoto algorithm for rate-distortion (covered in lossy/lossy1-foundations.html):

  1. Initialize $p^{(0)}(t|x)$ (e.g., random soft assignment).
  2. Compute $p^{(k)}(t) = \sum_x p(x)\, p^{(k)}(t|x)$.
  3. Compute $p^{(k)}(y|t) = \sum_x p(y|x)\, p^{(k)}(x|t)$.
  4. Update $p^{(k+1)}(t|x) \propto p^{(k)}(t)\,\exp\!\bigl(-\beta\, D_{\mathrm{KL}}(p(y|x)\|p^{(k)}(y|t))\bigr)$.
  5. Repeat until convergence.

Convergence to a local minimum is guaranteed by the same double-minimization argument as Blahut-Arimoto: each step monotonically decreases $\mathcal{L}_{\mathrm{IB}}$.

§03 — IB as rate-distortion

Relevance decomposition proof detail

We expand:

$$\mathbb{E}_{p(x,t)}[D_{\mathrm{KL}}(p(y|x)\|p(y|t))] = \sum_{x,t} p(x,t)\sum_y p(y|x)\log\frac{p(y|x)}{p(y|t)}.$$

Split the log:

$$= \sum_{x,t} p(x,t)\left[-H(Y|X=x) + H_{p(y|x)}(p(y|t))\right]$$

where $H_{p(y|x)}(p(y|t)) = -\sum_y p(y|x)\log p(y|t)$ is the cross-entropy.

The first term: $\sum_{x,t} p(x,t)[-H(Y|X=x)] = -H(Y|X)$ (marginalizing over $t$).

The second term:

$$-\sum_t p(t)\sum_y \left[\sum_x p(x|t)\, p(y|x)\right]\log p(y|t) = -\sum_t p(t)\sum_y p(y|t)\log p(y|t) = H(Y|T).$$

Total: $-H(Y|X) + H(Y|T) = I(X;Y) - I(T;Y)$.

Properties of the log-loss distortion

  • $d_{\mathrm{IB}}(x,t) \ge 0$ with equality iff $p(y|x) = p(y|t)$.
  • Not symmetric: $d_{\mathrm{IB}}(x,t) \ne d_{\mathrm{IB}}(t,x)$ in general (KL divergence is asymmetric).
  • The distortion depends on the encoder through $p(y|t)$, making the IB R(D) problem non-standard: the distortion itself is a function of the optimization variable.

§04 — The information plane

Properties of the IB curve

The IB curve $I_{\mathrm{IB}}(R)$ as a function of rate $R = I(X;T)$:

  • Concavity: follows from the concavity of MI in the conditional distribution (the set of achievable $(R, I_Y)$ pairs is convex; the upper boundary of a convex set is concave).
  • Monotonicity: increasing $R$ allows more expressive encoders, so $I_{\mathrm{IB}}(R)$ is non-decreasing.
  • Slope: at the optimal point for a given $\beta$, the slope of the IB curve is $1/\beta$ (standard Lagrangian duality).

Phase transitions — critical $\beta_1$

For $\beta < \beta_1$, the only solution is the trivial one: $p(t|x) = p(t)$ for all $x$ (i.e., $T \perp X$).

At $\beta = \beta_1$, this solution becomes unstable and a non-trivial solution bifurcates. The critical value can be computed by linearizing the self-consistent equations around the trivial solution:

$$\beta_1 = \frac{1}{\lambda_{\max}(M)},$$

where $M$ is the matrix with entries $M_{yy'} = \sum_x p(x)\, p(y|x)\, p(y'|x) / p(y)$ and $\lambda_{\max}$ is its largest non-trivial eigenvalue. This is related to the $\chi^2$ divergence between $p(y|x)$ and $p(y)$ averaged over $x$.

DPI monotonicity for DNN layers

For deterministic layers $T_l = f_l(T_{l-1})$, the Markov chain $Y - X - T_1 - \cdots - T_L$ gives:

$$I(X;Y) \ge I(T_1;Y) \ge I(T_2;Y) \ge \cdots \ge I(T_L;Y).$$

Caveat: for deterministic functions of continuous variables, $I(X;T_l) = H(T_l)$ can be infinite. The discrete-input case is clean; the continuous case requires careful regularization or binning — a key challenge addressed by the Variational IB in Lecture 2.
