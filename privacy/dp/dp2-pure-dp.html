01

Randomized Response

A 1960s survey trick that already realizes local DP.

Randomized Response (RR)

Each of $n$ users holds a sensitive bit $D_i \in \{0,1\}$ and reports

$$Y_i \;=\; \begin{cases} D_i & \text{w.p. } \tfrac12 + \gamma \\ 1 - D_i & \text{w.p. } \tfrac12 - \gamma \end{cases}, \qquad 0 < \gamma \le \tfrac12.$$
  • $\gamma = \tfrac12$: no flipping — no privacy (true bit released).
  • $\gamma = 0$: full flipping (uniform output) — perfect privacy, zero utility.
  • $\gamma$ is the privacy/utility knob.

Unbiased Estimate of $D_i$

Mean of the response:

$$\mathbb{E}[Y_i] \;=\; D_i \bigl(\tfrac12+\gamma\bigr) + (1-D_i)\bigl(\tfrac12-\gamma\bigr) \;=\; 2\gamma D_i + \tfrac12 - \gamma.$$

Solving for $D_i$ yields the debiased estimator

$$\hat D_i \;=\; \frac{Y_i - \tfrac12 + \gamma}{2\gamma}, \qquad \mathbb{E}[\hat D_i] = D_i.$$

Estimating the Fraction $p$

The curator wants to release

$$p \;=\; \frac{1}{n}\sum_{i=1}^n D_i \quad\text{privately} \;\Longrightarrow\; \hat p = \frac{1}{n}\sum_{i=1}^n \frac{Y_i - \tfrac12 + \gamma}{2\gamma}.$$
  • $\hat p$ is unbiased: $\mathbb{E}[\hat p] = p$.
  • Variance bound (independent $Y_i \in \{0,1\}$):
$$\mathrm{Var}[\hat p] \;=\; \frac{1}{4 n^2 \gamma^2}\sum_{i=1}^n \mathrm{Var}[Y_i] \;\le\; \frac{1}{16 n \gamma^2}.$$

Sample Complexity via Chebyshev

Chebyshev's inequality.

For $\mathbb{E}[X] = \mu$, $\mathrm{Var}(X) = \sigma^2$, $\Pr[|X - \mu| > k\sigma] \le 1/k^2$.

Applied to $\hat p$ with $\sigma = 1/(4\gamma\sqrt n)$:

$$\Pr\!\left[|\hat p - p| > \frac{k}{4\gamma\sqrt n}\right] \;\le\; \frac{1}{k^2}.$$

To guarantee $\Pr[|\hat p - p| > \alpha] \le \beta$ take $\alpha = k/(4\gamma\sqrt n)$, $\beta = 1/k^2$:

$$n \;=\; \frac{1}{16\, \gamma^2\, \beta\, \alpha^2}.$$

Trade-off Summary

Accuracy $\alpha \downarrow$

Tighter $\hat p \approx p$ band requires $n \propto 1/\alpha^2$.

Confidence $\beta \downarrow$

Stronger probability guarantee needs $n \propto 1/\beta$.

Privacy $\gamma \downarrow$

More privacy needs $n \propto 1/\gamma^2$.

Three knobs: accuracy, confidence, privacy — all priced in sample size.
02

$\varepsilon$-Differential Privacy

A formal, worst-case privacy guarantee.

Neighboring Datasets

Datasets $D, D' \in \mathcal{X}^n$ are neighbors if they differ in one individual's record:

$$D = (X_1, \dots, X_i, \dots, X_n), \quad D' = (X_1, \dots, X_i', \dots, X_n).$$
  • Replace: change one record (used in this course).
  • Add / delete: insert or remove one record. Replace = delete + insert, so the variants are essentially equivalent under composition.

$\varepsilon$-Differential Privacy

Definition (Dwork–McSherry–Nissim–Smith, 2006).

A randomized mechanism $\mathcal{M}: \mathcal{X}^n \to \mathcal{Y}$ is $\varepsilon$-DP if for every pair of neighboring datasets $D, D'$ and every measurable $O \subseteq \mathcal{Y}$,

$$\Pr[\mathcal{M}(D) \in O] \;\le\; e^{\varepsilon}\, \Pr[\mathcal{M}(D') \in O].$$

By symmetry $e^{-\varepsilon} \le \Pr[\mathcal{M}(D) \in O]/\Pr[\mathcal{M}(D') \in O] \le e^{\varepsilon}$.

Dwork, McSherry, Nissim, Smith, "Calibrating Noise to Sensitivity in Private Data Analysis", TCC 2006. Test-of-time TCC 2016; Gödel Prize 2017.

Why Randomization is Necessary

Example.

Let $f(D) = \tfrac{1}{n}\sum_i X_i$ and $\mathcal{M}(D) = f(D)$. Then for neighboring $D, D'$,

$$\mathcal{M}(D') = f(D) + \frac{X_i' - X_i}{n},$$

so observing both reveals $X_i' - X_i$ exactly and identifies which dataset is which.

Any deterministic $\mathcal{M}$ has the same problem. DP requires randomization.

Notes on the Definition

  • $\varepsilon = 0$ ⇒ perfect privacy; smaller $\varepsilon$ ⇒ stronger privacy.
  • Worst case: must hold for every neighboring $(D, D')$ and every event $O$.
  • Multiplicative: additive-slack-only definitions fail (see Lecture 3).
  • $e^{\varepsilon} \approx 1 + \varepsilon$ for small $\varepsilon$; exponential form simplifies composition.
  • DP is a notion of similarity between distributions $\mathcal{M}(D)$ and $\mathcal{M}(D')$ — alternative divergences yield related definitions (KL-DP, RDP, $f$-DP).

Hypothesis-Testing Interpretation

Adversary chooses an event $O$ and tests

  • $H_0$: data is $D$   vs.   $H_1$: data is $D'$;
  • declares $H_1$ if $\mathcal{M}(D) \in O$.

Let $q$ = false-positive rate, $p$ = false-negative rate. $\varepsilon$-DP implies

$$1 \le q + p\, e^{\varepsilon}, \qquad 1 \le p + q\, e^{\varepsilon}.$$

Both errors cannot be small simultaneously — privacy = indistinguishability.

Aside: False Positive vs False Negative

Example (Covid test).

Covid patient: $T \sim \mathrm{Lap}(39, b)$; healthy: $T \sim \mathrm{Lap}(38, b)$. Best test: declare Covid if $T > 38.5$.

  • False positive (FP): healthy person flagged as Covid.
  • False negative (FN): Covid patient declared healthy.

Reducing one inflates the other — the same trade-off DP imposes on adversaries.

RR Satisfies $4\gamma$-DP

Let $f(D) = D$ and $\mathcal{M}(D) = Y = (Y_1, \dots, Y_n)$ from RR. For neighbors $D, D'$ differing at index $i$:

$$\frac{\Pr[\mathcal{M}(D) = y]}{\Pr[\mathcal{M}(D') = y]} \;=\; \frac{\Pr[Y_i = y_i \mid D_i]}{\Pr[Y_i = y_i \mid D_i']} \;\le\; \frac{\tfrac12+\gamma}{\tfrac12-\gamma} \;\le\; e^{4\gamma}.$$

Hence RR is $\varepsilon$-DP with $\varepsilon = 4\gamma$. As $\gamma \to 0$, $\varepsilon \to 0$ (more privacy, more noise).

Exercise: extend the bound to arbitrary measurable $O$.

03

Laplace Mechanism

$\varepsilon$-DP for numeric queries via noise calibrated to sensitivity.

$\ell_1$ Sensitivity

Definition.

For $f : \mathcal{X}^n \to \mathbb{R}^k$, the $\ell_1$-sensitivity is

$$\Delta_1 \;=\; \max_{D, D' \text{ neighbors}} \|f(D) - f(D')\|_1.$$

Large $\Delta_1$ ⇒ one record can move $f$ a lot ⇒ more noise needed.

Example.

For $f(D) = \tfrac{1}{n}\sum_{i} X_i$ with $X_i \in [a, b]$, $\Delta_1 = (b-a)/n$.

Laplace Mechanism

Mechanism (Dwork et al. 2006).

Release $\mathcal{M}(D) = f(D) + (Y_1, \dots, Y_k)$ where $Y_i \overset{\text{iid}}\sim \mathrm{Lap}(0, \Delta_1/\varepsilon)$. Then $\mathcal{M}$ is $\varepsilon$-DP.

Laplace pdf: $f_{Y}(y) = \frac{1}{2b}\exp(-|y|/b)$ with scale $b = \Delta_1/\varepsilon$.

Noise scale $\Delta_1/\varepsilon$ hides any single-record change of magnitude $\le \Delta_1$.

Proof — Density Ratio

For neighbors $D, D'$ and any $z \in \mathbb{R}^k$:

$$f_{\mathcal{M}(D)}(z) = \prod_{i=1}^k \frac{1}{2b}\exp\!\Bigl(-\frac{|z_i - f(D)_i|}{b}\Bigr), \quad f_{\mathcal{M}(D')}(z) \text{ similarly.}$$

Ratio:

$$\frac{f_{\mathcal{M}(D)}(z)}{f_{\mathcal{M}(D')}(z)} = \prod_i \exp\!\Bigl(\frac{|z_i - f(D')_i| - |z_i - f(D)_i|}{b}\Bigr) \le \exp\!\Bigl(\frac{\|f(D) - f(D')\|_1}{b}\Bigr).$$

By definition $\|f(D) - f(D')\|_1 \le \Delta_1$, so ratio $\le e^{\Delta_1/b} = e^{\varepsilon}$. Integrating over any event $T$ preserves the bound. $\blacksquare$

Noise Variance

  • $\mathrm{Var}(Y_i) = 2 b^2 = 2 \Delta_1^2 / \varepsilon^2$.
  • Large $\Delta_1$ ⇒ more sensitive $f$ ⇒ louder noise.
  • Small $\varepsilon$ ⇒ tighter privacy ⇒ louder noise.
Two equivalent intuitions: noise hides a single record's worst-case impact; noise budget scales with the function's worst-case influence.
04

DP Selection

Exponential mechanism & noisy max for non-numeric outputs.

Selection Problems

Voting.

$d$ candidates, $n$ users, $X_i \subseteq \{1, \dots, d\}$. Score: $q(j; D) = |\{i: j \in X_i\}|$. Non-private: $\arg\max_j q(j; D)$.

Pricing.

Set price $p$ given thresholds $X_i$. Revenue $q(p; D) = p \cdot |\{i: p \le X_i\}|$. Adding Laplace noise to $p$ can crash revenue to $0$.

Both are selection over a discrete (or discretized) set — Laplace on the output is wrong. Need a DP mechanism that respects the discrete structure.

Exponential Mechanism

Definition.

Given utility $q : \mathcal{X}^n \times \mathcal{Y} \to \mathbb{R}$ with sensitivity $\Delta = \max_{D, D'} \max_y |q(D, y) - q(D', y)|$, the Exponential Mechanism outputs

$$\Pr[\mathcal{M}_{\mathrm{EM}}(D) = y] \;\propto\; \exp\!\Bigl(\frac{\varepsilon\, q(D, y)}{2 \Delta}\Bigr).$$

$\mathcal{M}_{\mathrm{EM}}$ is $\varepsilon$-DP. High-utility outputs are exponentially more likely; the $2\Delta$ scaling absorbs single-record perturbations.

Exponential Mechanism — Utility (High Probability)

Theorem.

Let $q_{\max} = \max_y q(D, y)$ and $d = |\mathcal{Y}|$. For $t \ge 0$,

$$\Pr\!\Bigl[q(D, Y) < q_{\max} - \tfrac{2\Delta(\log d + t)}{\varepsilon}\Bigr] \;\le\; e^{-t}.$$

Proof.

Let $B_t = \{y : q(D, y) < q_{\max} - \tfrac{2\Delta(\log d + t)}{\varepsilon}\}$. Then

$$\Pr[Y \in B_t] \le \tfrac{\sum_{y\in B_t} \exp(\tfrac{\varepsilon}{2\Delta} q(y;D))}{\exp(\tfrac{\varepsilon}{2\Delta} q_{\max})} \le |B_t|\, e^{-(\log d + t)} \le e^{-t}. \;\blacksquare$$

Exponential Mechanism — Utility (Expectation)

Theorem.

$\mathbb{E}[q_{\max} - q(D, Y)] \;\le\; \tfrac{2\Delta}{\varepsilon} (\log d + 1)$.

Proof.

Let $Z = \tfrac{\varepsilon}{2\Delta}(q_{\max} - q(D, Y))$. With $z = \log d + t$,

$$\mathbb{E}[Z] = \int_0^\infty \Pr[Z > z]\, dz \le \log d + \int_0^\infty e^{-t}\, dt = \log d + 1. \;\blacksquare$$

Logarithmic in $|\mathcal{Y}|$ — well-suited for large finite output spaces.

Example: Most-Frequent Item (Voting)

  • $\Delta = 1$ (one voter changes one count by at most 1).
  • $d = 100$, $\varepsilon = 0.5$, target $\beta = e^{-t} = 0.01$ ⇒ $t = \ln 100 \approx 4.6$.
  • Utility gap bound: $\tfrac{2\Delta}{\varepsilon}(\log d + t) \approx 36.8$.
$$\Pr[q_{\max} - q(D; Y) > 36.8] \le 0.01.$$

If winner leads runner-up by $> 36.8$ votes, winner is selected w.p. $\ge 99\%$.

Noisy Max (Report Noisy Max)

Definition.

Let $q$ have sensitivity $\Delta$. For each $y \in \mathcal{Y}$:

  1. Sample $Z_y \sim \mathrm{Exp}(2\Delta/\varepsilon)$ (i.i.d.).
  2. Compute $s_y = q(y; D) + Z_y$.
  3. Output $y^* = \arg\max_y s_y$.

Same flavor as the exponential mechanism, but operates on noisy scores directly.

Theorem — Noisy Max is $\varepsilon$-DP

Proof sketch.

For neighbors $X, X'$ and any $y$, condition on $Z^{d\setminus y} = z^{d\setminus y}$:

$$\Pr[\mathcal{M}_{\mathrm{NM}}(X) = y \mid z^{d\setminus y}] \;=\; \Pr\!\bigl[Z_y \ge \max_{j \ne y} q(j;X) - q(y;X) + z_j\bigr].$$

$|q(j;X) - q(j;X')| \le \Delta$ and similarly for $q(y;\cdot)$ shifts the threshold by at most $2\Delta$. The key inequality for $Z_y \sim \mathrm{Exp}(2\Delta/\varepsilon)$ is

$$\Pr[Z_y \ge K - 2\Delta] \;\le\; e^{\varepsilon}\, \Pr[Z_y \ge K].$$

Integrating over $z^{d\setminus y}$ yields $\Pr[\mathcal{M}_{\mathrm{NM}}(X) = y] \le e^{\varepsilon} \Pr[\mathcal{M}_{\mathrm{NM}}(X') = y]$. $\blacksquare$

Noisy Max — Utility

Theorem.

$\Pr\!\bigl[q_{\max} - q(Y; X) > \tfrac{2\Delta(\log d + t)}{\varepsilon}\bigr] \le e^{-t}$.

Proof.

Let $c = \tfrac{2\Delta(\log d + t)}{\varepsilon}$. By optimality, $q(Y; X) + Z_Y \ge q_{\max} + Z_{y^*}$ so $q_{\max} - q(Y; X) \le Z_Y - Z_{y^*}$. Union bound + exponential tail:

$$\Pr[\exists j: Z_j - Z_{y^*} > c] \;\le\; d \cdot \exp\!\bigl(-\tfrac{\varepsilon c}{2\Delta}\bigr) \;=\; e^{-t}. \;\blacksquare$$

Same logarithmic regret as the exponential mechanism.

Recap

  • Randomized response: $4\gamma$-DP via per-user bit flipping; sample complexity $\Theta(1/(\gamma^2 \alpha^2 \beta))$.
  • $\varepsilon$-DP: $\Pr[\mathcal{M}(D)\in O] \le e^\varepsilon \Pr[\mathcal{M}(D')\in O]$ for all neighboring $D, D'$.
  • Laplace mechanism: $\Delta_1/\varepsilon$-scale noise on numeric queries.
  • Exponential mechanism & Noisy Max: DP selection over discrete $\mathcal{Y}$ with $O(\log |\mathcal{Y}|/\varepsilon)$ regret.
